JavaScript Spiel: Codebreaker


Ziel ist es, einen dreistelligen Zahlencode zu erraten (wie ein Vorhängeschloss). Der Zahlencode wird zufällig generiert. Du hast 12 Versuche, um die richtige dreistellige Kombination zu erraten.

ANLEITUNG: Drücke "Start" um das Spiel zu beginnen. Mit "Code überprüfen" kannst du den jeweiligen Code überprüfen. Wenn dieser mit der Zufallszahl übereinstimmt, hast du gewonnen und erhältst eine Nachricht. Wenn du noch eine Runde spielen willst, drücke auf "Neues Spiel".
